# 3-Month AI Intern Syllabus

## Goal
Train beginner AI interns from zero to a level where they can understand core AI concepts, build small Python utilities, create a basic RAG chatbot, and contribute to simple agent-based workflows.

## Training Format
- Duration: 12 weeks
- Pace: Full-time internship
- Focus: Learning by doing
- Outcome: A working capstone project, weekly mini-projects, and strong fundamentals

## Learning Outcomes
By the end of 3 months, the intern should be able to:
- Use Python comfortably for basic automation and data handling
- Work with lists, dictionaries, functions, files, and simple data structures
- Understand basic AI and LLM concepts
- Explain and build a simple RAG pipeline
- Understand what an AI agent is and how tool use works
- Build and present a small AI project with documentation

## Month 1: Foundations

### Week 1: Orientation and Python Setup
**Topics**
- Introduction to the internship
- How AI teams work
- Python installation and environment setup
- VS Code, Jupyter Notebook, terminal basics
- Virtual environments and `pip`

**Practice**
- Install Python and set up the environment
- Run first Python scripts
- Print, input, variables, and basic types

**Deliverable**
- Working development setup
- First Python notebook or script

### Week 2: Python Basics and Data Handling
**Topics**
- Lists, tuples, dictionaries, sets
- Loops and conditional statements
- Functions and return values
- String handling
- Reading and writing files
- JSON and CSV basics
- Intro to NumPy
- Arrays vs lists
- Simple data cleaning

**Practice**
- Write small programs using lists and dictionaries
- Solve simple logic problems
- Load a CSV file
- Convert data to JSON
- Use NumPy for simple numeric operations

**Deliverable**
- 5 to 8 small Python exercises
- Small data processing script

### Week 3: NLP Series
**Topics**
- What is NLP?
- Text preprocessing
- Tokenization
- Stop words, stemming, and lemmatization
- Part-of-speech tagging
- Named entity recognition
- Text classification basics
- Intro to common NLP libraries

**Practice**
- Clean and preprocess sample text
- Tokenize text using a library
- Try a simple text classification or sentiment example

**Deliverable**
- NLP notebook with preprocessing examples

### Week 4: Git, APIs, and AI Introduction
**Topics**
- Git and GitHub basics
- Commit, push, pull, branches
- What is an API?
- Calling APIs with Python
- Intro to AI, ML, and LLMs
- Tokens, prompts, context windows

**Practice**
- Push code to GitHub
- Call a public API
- Experiment with prompts

**Deliverable**
- Simple API-based Python project

## Month 2: AI and RAG Foundations

### Week 5: LLM Basics
**Topics**
- What a language model does
- Prompting basics
- Temperature, top-p, and response behavior
- Hallucinations and limitations
- Prompt engineering basics

**Practice**
- Compare outputs with different prompts
- Test temperature effects

**Deliverable**
- Prompt experiment notes and examples

### Week 6: Embeddings and Vector Search
**Topics**
- What embeddings are
- Similarity search
- Vector databases overview
- Chunking documents
- Why retrieval matters

**Practice**
- Split sample documents into chunks
- Create embeddings for text
- Search similar chunks

**Deliverable**
- Mini document search demo

### Week 7: RAG Pipeline Basics
**Topics**
- RAG architecture
- Document loading
- Chunking strategy
- Embedding + retrieval flow
- Context assembly for the LLM

**Practice**
- Build a simple Q&A flow using documents
- Test different chunk sizes

**Deliverable**
- Basic RAG prototype

### Week 8: RAG Quality and Evaluation
**Topics**
- Common RAG failure modes
- Retrieval quality vs answer quality
- Relevance, accuracy, and faithfulness
- Adding citations or sources
- Debugging bad answers

**Practice**
- Evaluate answers on a small question set
- Improve retrieval with better chunking or prompts

**Deliverable**
- Improved RAG bot with evaluation notes

## Month 3: Agents, Safety, and Capstone

### Week 9: Agent Basics
**Topics**
- What an AI agent is
- Chatbot vs agent
- Tool use and function calling
- Planning and action loops
- When to use agents and when not to

**Practice**
- Build a small tool-using assistant
- Test simple tool calls

**Deliverable**
- Basic agent demo

### Week 10: Workflow, Memory, Safety, and Debugging
**Topics**
- Multi-step tasks
- Memory concepts
- State handling
- Task decomposition
- Guardrails for tool use
- Prompt injection basics
- Safe tool access
- Logging inputs and outputs
- Retry handling
- Basic monitoring and error tracking

**Practice**
- Create a workflow that uses multiple steps
- Add simple memory or state tracking
- Add logs to the project
- Test failure cases

**Deliverable**
- Workflow prototype with logging and safety checks

### Week 11: Capstone Project Build
**Topics**
- Choose a project idea
- Define scope and requirements
- Plan data flow and components
- Build the first working version
- Collect feedback and improve

**Practice**
- Start the capstone project
- Implement the main features
- Test the core workflow end to end

**Deliverable**
- Working capstone prototype

### Week 12: Capstone Finalization and Presentation
**Topics**
- Project finalization
- README writing
- Demo preparation
- Result presentation
- Retrospective and next steps

**Capstone Options**
- Company document Q&A bot
- Internal AI assistant for FAQs
- Small agent that automates a task workflow

**Deliverable**
- Final project demo
- GitHub repository
- Short presentation

## Weekly Routine
- 1 day: theory and guided learning
- 2 days: coding exercises
- 1 day: project work
- 1 day: review, feedback, and documentation

## Assessment Criteria
- Learning attitude
- Ability to ask good questions
- Code quality for beginner level
- Progress week to week
- Project completion
- Communication and documentation

## Suggested Checkpoints
- End of Month 1: Python and tools comfort
- End of Month 2: Working RAG prototype
- End of Month 3: Capstone project and presentation

## Final Outcome
At the end of the 3 months, the intern should be able to:
- Understand the basics of AI engineering work
- Build simple Python-based tools
- Explain and implement a beginner RAG system
- Understand agent workflows
- Work on real AI tasks with guidance
