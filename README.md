# AI Internship Learning Journey (12 Weeks)

## Overview

This repository documents my learning journey through a 12-week AI Internship training program. The program starts with Python fundamentals and gradually progresses toward modern AI concepts such as Natural Language Processing (NLP), Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), AI agents, and workflow automation.

The learning approach throughout the internship is project-based. Each week focuses on specific concepts, practical exercises, and a small deliverable that builds upon the previous week's knowledge. Every week's work is organized in its own folder with source code, notebooks, documentation, and a dedicated README.

## Learning Goals

The primary objectives of this internship are to:

* Build strong Python programming fundamentals
* Learn data handling using Python
* Understand Natural Language Processing concepts
* Learn how APIs work and interact with external services
* Understand the fundamentals of AI, Machine Learning, and Large Language Models
* Learn embeddings and semantic search
* Build Retrieval-Augmented Generation (RAG) systems
* Understand AI agents, workflows, memory, and tool usage
* Complete an end-to-end AI capstone project

## Challenges and Learning Experience

Throughout this internship, I encountered several challenges while working on different weekly projects. Rather than treating them as obstacles, I used them as opportunities to improve my understanding of Python, AI concepts, and software development practices.

Some of the key challenges included:

* Understanding the differences between Python data structures and choosing the appropriate one for different tasks.
* Reading and processing CSV and JSON files while handling file path and directory issues.
* Learning how to debug Python programs using error messages and stack traces.
* Working with third-party APIs and handling common issues such as deprecated endpoints, network errors, and invalid responses.
* Understanding Natural Language Processing concepts such as tokenization, stemming, lemmatization, and named entity recognition.
* Grasping the concept of embeddings, semantic similarity, and how vector search differs from traditional keyword search.
* Understanding the complete Retrieval-Augmented Generation (RAG) pipeline, from document chunking and embedding generation to retrieval and answer generation.
* Learning when to use AI agents instead of traditional chatbots, and understanding concepts such as tool calling, workflows, memory, logging, and safety.
* Writing clear documentation and organizing projects in a maintainable structure using Git and GitHub.

Each challenge strengthened my problem-solving skills and reinforced the importance of reading documentation, experimenting with different approaches, debugging systematically, and continuously improving my implementations.
